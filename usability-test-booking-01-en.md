# Usability Test Report

## General Information
- Website: https://www.booking.com
- Test date: 2026-02-14
- User profile: Elderly user (65+), low technical skills

## Test Goal
Evaluate how easy it is for an elderly user to find a hotel with free cancellation.

## Test Scenario
User searches for a hotel in Tel Aviv for 2 nights and checks free cancellation option.

## Issues Found
1. Free cancellation filter is not easy to find for elderly users.
2. Text size in filters and descriptions is too small and causes eye strain.
3. There is no visible information about special offers for users aged 65+.
   
## What Worked Well
- Logical and well-structured placement of information.
- Availability of customer reviews helps decision-making.

## Overall Impression
[ ] Easy to use  
[x] Some difficulties  
[ ] Difficult to use for elderly users

## Recommendations
- Increase font size in filters and key information sections.
- Make the free cancellation filter more visible.
- Add or clearly indicate special offers for elderly users (65+).
