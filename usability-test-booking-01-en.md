# Usability Test Report

## General Information
- Website: https://www.booking.com
- Test date: 2026-02-14
- User profile: Elderly user (65+), low technical skills

## Test Goal
Evaluate how easy it is for an elderly user to find a hotel with free cancellation.

## Test Scenario
User searches for a hotel in Tel Aviv for 2 nights and checks free cancellation option.

## Issues Found
1. Text size of filters is small and difficult to read.
2. Too many filters on the left side cause confusion.
3. Free cancellation option is not clearly highlighted.

## What Worked Well
- Search field is easy to find.
- Calendar selection is clear.

## Overall Impression
[ ] Easy to use  
[x] Some difficulties  
[ ] Difficult to use for elderly users

## Recommendations
- Increase font size for filters.
- Highlight free cancellation option more clearly.
